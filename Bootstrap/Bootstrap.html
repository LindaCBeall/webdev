    Bootstrap 4

    alpha - first release, very buggy beta - a little more polished but still with assumption that things might change or have "Breaking Changes" stable bootstrap's first alpha 4.0.0 released in 2015, stable version not released until 2018 (during that time were releasing beta and stable versions of 3.)
    For changes, you can either click drop down v4.5 to view the documentation for older versions (the entire site will change) OR click the Migration side menu item
  • Flexbox enabled by default (instead of floats)
  • Bootstrap primary CSS unit changed from px to rem
  • Global font size from 14px to 16px
  • Added 5th grid size/screen size tier, no longer a "xs", just - (col-, d-, etc)
  • Utility classes:
    Display utility d-{breakpoint}-{value} always controls from breakpoint up d- is all sizes d-none d-lg-inline will have none up until lg, then do inline from there after (unless specify different for xl) May have text, photos that are too much for small screen and want to hide them
  • Flexbox - not exclusive to BS but is new to BS4
  • Navs - rewrote with flexbox, dont need li anymore, can do with a tags and classes
  • cards
  • no built in glyphicon - use fontawesome
